# Estudo preliminar

## 1. Descrición do proxecto

O proxecto consiste na mellora dunha aplicación feita previamente no IES San cleemnte donde se implementan un novo conxunto de funcionalidades e melloras na experiencia de usuario e funcionalidades existentes.

### 1.1. Xustificación do proxecto

Considerase un proxecto/tarefa feito en clase que tiña potencial para ser máis extenso

### 1.2. Funcionalidades do proxecto

- A base do xogo matamarcianos, pero con funcionalidades novas e melloradas.
- Correción de bugs
- Menu de pausa
- Soporte para diferentes resolucións de pantalla
- Soporte para diferentes idiomas

### 1.3. Estudo de necesidades

Existen numerosa aplciación de esta tematica, pero esta orientada a aprender en maior profundade o motor Unity e a programación en C#.

### 1.4. Persoas destinatarias

Toda persoa con acceso a un equipo con sistema operativo linux, Windows ou executalo via Web

### 1.5. Modelo de negocio

O modelo de negocio do proxecto será a distribución totalmente gratuíta da aplicación e sen publicidade.

## 2. Requirimentos
Unity
